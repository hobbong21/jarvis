# S.A.R.V.I.S ‚Äî Personal AI Assistant

A multimodal AI assistant inspired by the 4-stage agent pattern (Task Planning ‚Üí Model Selection ‚Üí Task Execution ‚Üí Response Generation). Features face recognition, voice interaction, and tool-augmented intelligence.

## Architecture

- **Backend**: FastAPI + WebSockets (`server.py`) on port 5000
- **Frontend**: HTML5 Canvas / Vanilla JS / CSS in the `web/` directory
- **AI Brain**: Claude (Anthropic) or Ollama as LLM backend (`brain.py`)
- **STT**: Faster-Whisper for speech-to-text
- **TTS**: Edge-TTS (Microsoft) for text-to-speech
- **Vision**: OpenCV + optional face_recognition for webcam analysis

## Key Files

- `server.py` ‚Äî FastAPI web server (entry point for web mode)
- `brain.py` ‚Äî LLM controller (Claude tool_use loop + Ollama simple chat)
- `tools.py` ‚Äî Tool definitions and executor (web search, weather, timer, memory, vision)
- `audio_io.py` ‚Äî Speech recording, Whisper STT, Edge-TTS
- `vision.py` ‚Äî VisionSystem (desktop) + WebVision (web, browser-pushed frames)
- `config.py` ‚Äî Centralized configuration via environment variables
- `auth.py` ‚Äî User authentication with PBKDF2 hashing
- `emotion.py` ‚Äî 7 assistant emotional states
- `web/` ‚Äî Frontend assets (HTML, CSS, JS, orb animation)

## Environment Variables

- `ANTHROPIC_API_KEY` ‚Äî Required for Claude backend (set as a secret)
- `OPENAI_API_KEY` ‚Äî Required for OpenAI backend and `compare` mode (set as a secret)
- `GOOGLE_API_KEY` ‚Äî Required for Gemini backend (set as a secret). `GEMINI_API_KEY` is honoured as a legacy fallback. Default model `gemini-2.5-flash` (override with `GEMINI_MODEL`).
- `SARVIS_BACKEND` ‚Äî `"openai"` (default), `"claude"`, `"ollama"`, `"zhipuai"`, `"gemini"`, or `"compare"`
- `OLLAMA_HOST` ‚Äî Ollama server URL (default `http://localhost:11434`). Use a tunnel URL (e.g. cloudflared) when SARVIS runs on Replit but Ollama runs on a local machine.
- `OLLAMA_MODEL` ‚Äî Ollama model tag (default `qwen2.5:7b`). Examples: `llama3.2:3b`, `qwen2.5:14b`, `gemma2:9b`.
- `PORCUPINE_ACCESS_KEY` ‚Äî Optional, for desktop wake-word detection

## Running

The workflow runs `python server.py` on port 5000. The app auto-loads the Whisper model on startup.
